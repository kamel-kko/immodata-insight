ImmoData - Politique de confidentialite

Derniere mise a jour : 5 avril 2026

En resume : ImmoData ne collecte aucune donnee personnelle. Toutes les informations restent dans votre navigateur. Rien n'est envoye a des serveurs tiers.

1. Donnees collectees

ImmoData ne collecte aucune donnee personnelle. L'extension :

  • Ne cree pas de compte utilisateur
  • Ne demande pas d'adresse email
  • Ne suit pas votre navigation
  • Ne place pas de cookies
  • N'utilise pas de service d'analytics externe (Google Analytics, Mixpanel, etc.)

2. Stockage local

ImmoData utilise le stockage local de Chrome (chrome.storage.local) pour enregistrer :

  • Cache API : les reponses des APIs Open Data sont mises en cache localement pour eviter les requetes repetees et accelerer l'affichage. Chaque entree a une duree de vie limitee (1 a 30 jours selon l'API).
  • Parametres : vos preferences (apport, duree de credit, cle API OpenRouteService si configuree).
  • Statistiques d'utilisation : compteurs locaux (nombre d'annonces analysees, de pages vues). Ces statistiques ne quittent jamais votre navigateur.
  • Suivi des annonces : historique de prix des annonces visitees pour detecter les baisses de prix.

Toutes ces donnees sont effacables a tout moment depuis la popup de l'extension (boutons "Vider le cache" et "Reinitialiser les stats").

3. APIs interrogees

Pour enrichir les annonces, ImmoData interroge exclusivement des APIs Open Data publiques francaises. Voici la liste complete :

API Donnees Operateur
Base Adresse Nationale Geocodage (adresse vers GPS) data.gouv.fr
DVF (OpenDataSoft) Transactions immobilieres public.opendatasoft.com
Georisques Risques naturels et industriels georisques.gouv.fr
Annuaire Education Ecoles et etablissements data.education.gouv.fr
Overpass (OpenStreetMap) Commerces et transports overpass-api.de
ADEME Diagnostics energetiques (DPE) data.ademe.fr
INSEE Donnees demographiques api.insee.fr
Merimee Monuments historiques data.culture.gouv.fr
RTE Lignes haute tension opendata.reseaux-energies.fr
OpenRouteService Isochrones (optionnel) openrouteservice.org

Les seules informations transmises a ces APIs sont les coordonnees GPS et le code INSEE derives de l'adresse du bien consulte. Aucune information personnelle n'est envoyee.

4. Liens d'affiliation

ImmoData peut afficher des suggestions de services partenaires (courtiers, diagnostiqueurs, assurances) sous forme de boutons dans le dashboard. Ces liens contiennent des parametres UTM pour identifier la source.

  • Les liens s'ouvrent dans un nouvel onglet uniquement si vous cliquez dessus
  • Aucun tracking entre les sites n'est effectue par ImmoData
  • Les impressions et clics sont comptes localement pour les statistiques internes

5. Permissions Chrome

Permission Usage
storage Sauvegarder le cache, les parametres et les statistiques locales
alarms Programmer le nettoyage automatique du cache expire
activeTab Lire le contenu de l'onglet actif pour extraire les donnees de l'annonce

6. Vos droits

Puisqu'aucune donnee personnelle n'est collectee ni transmise, il n'y a rien a supprimer cote serveur. Vous pouvez a tout moment :

  • Desactiver l'extension via le toggle dans la popup
  • Vider le cache et les statistiques depuis la popup
  • Desinstaller l'extension, ce qui supprime toutes les donnees locales

7. Contact

Pour toute question concernant cette politique de confidentialite, vous pouvez ouvrir une issue sur le depot GitHub du projet.
